The Scoutline Sandbox is a complete, isolated mirror of the production environment, designed for testing Agentic Lawyer integrations without affecting live case data.

## Accessing the Sandbox

- **Sandbox URL**: `https://sandbox.api.scoutline.com/v1`
- **Dashboard**: Toggle the "Sandbox" view in your Scoutline firm dashboard to retrieve test credentials.

## Simulated Data

The Sandbox environment provides high-fidelity, anonymized legal data including:
- **Mock Vault Documents**: High-density discovery files for indexing tests.
- **Simulated Agents**: Fully functional agentic workflows that generate "Sample Motion" drafts.

## Key Differences

| Feature | Production | Sandbox |
| :--- | :--- | :--- |
| **Data Retention** | Indefinite | 30 Days |
| **Filing Submission** | Live Court Systems | Simulated "Success" response |
| **MFA Requirements** | Mandatory for sensitive actions | Optional / Bypassable |

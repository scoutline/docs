The Scoutline SDK includes a pre-built UI library for React, Vue, and Vanilla JS, designed with "Interface Craft" precision.

## Pre-built Components

- **`<ScoutChatBox />`**: A full-sized chat interface with built-in citation rendering.
- **`<CitationPreview />`**: A high-density popup for displaying Source Map snippets.
- **`<VaultUploader />`**: A secure drag-and-drop area for case evidence.

## Implementation

<CodeGroup>
```typescript TypeScript
// React Example
import { ScoutChatBox } from '@scoutline/chat-sdk/react';

function App() {
  return <ScoutChatBox envId="env_xxx" />;
}
```


```go Go

```
</CodeGroup>

## Custom Styling

Every component can be styled using **Tailwind CSS** or standard **Vanilla CSS** tokens defined in your `configuration-api`.

For power users, the Scoutline Command Center provides a keyboard-driven interface to manage visual intelligence at lightning speed.

## The Command Bar

Press `Cmd + K` (Mac) or `Ctrl + K` (Windows) to open the Command Bar from anywhere on the Canvas.

### Common Commands

- `/node [name]`: Create a new entity node.
- `/link [a] to [b]`: Connect two existing nodes.
- `/go [event]`: Jump to a specific point on the Dynamic Timeline.
- `/audit`: Run a quick verifiability check on the current canvas state.

## Global Shortcuts

- `V`: Select tool.
- `L`: Link tool.
- `Space + Drag`: Pan across the high-density workspace.
- `Z + Scroll`: Zoom in/out of the node clusters.

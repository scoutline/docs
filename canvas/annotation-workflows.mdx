Scoutline allows for pixel-perfect annotations that bridge the gap between static evidence and moving strategy.

## Evidence Snippets

Highlight text in a Vault document and choose **"Pin to Canvas."** This creates an Annotation Node that:
- **Displays the Snippet**: Shows the verbatim text or image slice on the canvas.
- **Back-Links**: Clicking the node opens the full document in a side-pane at exactly the same location.

## Spatial Notes

Draw directly on the canvas using the **Ink Tool** to highlight specific clusters of nodes. These spatial notes are treated as a separate strategy layer and can be exported as part of the Evidence Map.
